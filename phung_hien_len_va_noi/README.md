Phụng Hiện Lên Và Nói là một bot voice dành cho Discord, giúp chuyển văn bản thành giọng đọc tiếng Việt do Ngọc Phụng xây dựng và phát triển.

-- Các lệnh sử dụng
!p-join → gọi bot vào voice channel của bạn
!pt <nội dung> → lệnh để bot nói

-- Ví dụ
!pt xin chào tôi là ngọc phụng


-------------------------------------------------------
"Phụng Hiện Lên Và Nói" is a voice bot for Discord that converts text into Vietnamese speech.

-- Available Commands
!p-join → summons the bot into your voice channel
!pt <text> → command to make the bot speak

-- Example
!pt xin chào tôi là ngọc phụng

--------------------------------------------------------
Thư viện sử dụng --- Libraries Used
pip install discord.py
pip install gTTS
pip install PyNaCl
